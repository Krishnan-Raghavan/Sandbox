\# Project Context: SEO Auditor MCP Server

\- This project builds a commercial-grade Model Context Protocol (MCP) Server.

\- Language/Runtime: TypeScript (Node.js) using `@modelcontextprotocol/sdk`.

\- Purpose: Audit structural technical SEO, extract content quality metrics, and generate Markdown/JSON remediation blueprints.

\- Target AI Interface: Optimized to handle requests seamlessly with Gemini 3.5 Flash.
